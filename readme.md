<h2>Usage:</h2>
<h3> To use it, go to Dashboard ==> General and scroll down to custom css and paste the following line</h3>

```css
@import url("https://cdn.jsdelivr.net/gh/FLIVLA/jellyfin@latest/default.css");
```

<h2>Regular:</h2>
<h3> To use it, got to Settings ==> Display, and scroll down to custom css and paste the following line</h3>

```css
@import url("https://cdn.jsdelivr.net/gh/FLIVLA/jellyfin@latest/regular.css");
```

<h2>Ultrawide</h2>
<h3> To use it, got to Settings ==> Display, and scroll down to custom css and paste the following line</h3>

```css
@import url("https://cdn.jsdelivr.net/gh/FLIVLA/jellyfin@latest/ultrawide.css");
```
